Hive DDL Commands:
1. Create a table:
CREATE TABLE employee (
  emp_id INT,
  emp_name STRING,
  emp_salary FLOAT
);

2. Describe a table:
DESCRIBE employee;

3. Alter table to add a new column:
ALTER TABLE employee ADD COLUMNS (emp_department STRING);

4. Drop a table:
DROP TABLE employee;

Hive DML Commands:
1. Insert data into a table:
INSERT INTO TABLE employee VALUES
  (1, 'John Doe', 50000, 'IT'),
  (2, 'Jane Smith', 60000, 'HR');

2. Select data from a table:
SELECT * FROM employee WHERE emp_salary > 55000;

DELETE FROM student WHERE student_id = 2;

update Student s set e=s.marks=50 where s.studentId=10;
